  <action>
Create a custom hook `useImageUpload` that handles image upload logic:

1. State management:
   - `uploading: boolean` - upload in progress
   - `progress: number` - 0-100 progress percentage
   - `error: string | null` - error message

2. `uploadFile(file: File)` function:
   - Validate file type (must be .png)
   - Validate file size (max 10MB)
   - Set `uploading: true`, start progress simulation
   - Call `fileService.uploadImage(file)` from existing service
   - Update progress during upload (use interval for simulation + track actual progress)
   - On success: return `StaticImage`, dispatch callback
   - On error: set error message, call `useToast.error`
   - Reset state after 300ms delay

3. Validation:
   - Show toast error for invalid format: "Only PNG files are supported"
   - Show toast error for file too large: "File size exceeds 10MB limit"

4. Return: `{ uploading, progress, error, uploadFile }`

Follow the pattern from `front-end/src/react/hooks/useFilters.ts` for error handling and state management.
Import `fileService` from `@/infrastructure/data/file-service`.
