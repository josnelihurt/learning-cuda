## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|

| md | 16px | Default element spacing |
| lg | 24px | Section padding |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: None

**Source:** Extended from existing `--spacing-xs: 8px, --spacing-sm: 12px, --spacing-md: 20px, --spacing-lg: 30px, --spacing-xl: 40px` in `/front-end/public/static/css/variables.css` to complete 8-point scale.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #FFF | Background, surfaces |
| Secondary (30%) | #FAFAFA | Cards, sidebar, nav |
| Accent (10%) | #ffa400 | Primary buttons, active tabs, selected items, hover states |
| Destructive | #d32f2f | Error states, stop button, destructive actions |

Accent reserved for: Primary buttons (Start Stream), active tabs, selected items, hover states, active connection indicators. Never for secondary actions or decorative elements.

**Source:** CSS variables from `/front-end/public/static/css/variables.css` (lines 9-32).

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Start Stream |
| Empty state heading | No Stream Active |

| Error state | {Specific error}: {Solution} (e.g., "Permission Denied: Please allow camera access in your browser settings") |
| Destructive confirmation | Stop Stream: Are you sure? This will disconnect the video stream and clean up all resources. |

**Source:** Derived from CONTEXT.md decisions D-11 (connection states), D-12 (toast notifications), D-13 (manual restart), and RESEARCH.md error handling patterns (lines 282-312).
